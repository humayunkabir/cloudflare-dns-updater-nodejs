# cloudflare-dns-updater-nodejs

To install dependencies:

```bash
bun install
```

To update environment variables:

```bash
cp .env.example .env
```

To run:

```bash
bun start
```

For corn:

```bash
bun cron
```

This project was created using `bun init` in bun v1.1.18. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
